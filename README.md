# CardKing1971 Customs

AI-Powered Trading Card Art Generator

## Overview

CardKing1971 Customs is an AI-powered trading card generation platform that allows users to create custom digital trading card artwork. The platform leverages advanced AI image generation models to transform trending topics and custom ideas into stunning trading card designs.

## Features

- **AI Trend Discovery** — Discover trending topics from social media and trading platforms powered by AI analysis.
- **Smart Prompts** — AI generates optimized prompts for stunning trading card artwork with one click.
- **Multiple AI Models** — Leverage Nano Banana, Nano Banana Pro, Grok Imagine, and OpenAI 4o models for high-quality image generation.
- **Multiple Variations** — Generate up to 10 unique variations of your trading card design in one batch.
- **Gallery & Favorites** — Save your creations, mark favorites, and easily recreate your best designs.
- **Print Layout** — Print-ready layouts for standard card sizes (2.5x3.5" and 4x6") on 8.5x11" paper.
- **Sticker Creator** — Create and print custom stickers from your card designs.
- **Non-Sports Creations** — Generate cards for pop culture, fantasy, sci-fi, anime, and more.
- **Card Back Designer** — Design custom card backs with patterns, text, and colors.
- **Export Options** — Export in PNG, JPG, or PDF format at standard, HD, or print-ready resolution.
- **Dark/Light Mode** — Toggle between dark and light themes.

## Tech Stack

- **Frontend:** React 19, TypeScript, Tailwind CSS, Framer Motion, Radix UI
- **Backend:** Express, tRPC, Drizzle ORM
- **Database:** MySQL / TiDB
- **AI Integration:** kie.ai API (Nano Banana models), OpenAI, Grok
- **Build:** Vite, esbuild

## Getting Started

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Set up environment variables
4. Run database migrations: `pnpm db:push`
5. Start development server: `pnpm dev`
